# GetOwnTasks - получение списка своих задач

Параметры

| Имя    | Тип | Обязательность |
|--------|-----|----------------|
| userId | JWT | +              |

## Алгоритм
1. Обогатиться задачами из **tasks** по **user_id = userId**
2. Вернуть ответ в соответсвии мапингу из [openApi](codest-task-openapi.yaml)
